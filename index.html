Coarse-to-Fine Prompt Enrichment for User-Friendly Controllable Speech Generation

AutoRich-TTS

Abstract

Text-prompted text-to-speech (TTS) has emerged as a novel paradigm for style-controllable speech synthesis, where textual descriptions serve as style prompts to govern the acoustic attributes of generated speech. However, attaining precise and high-quality stylistic control typically requires users to provide exhaustive fine-grained style descriptions, which incurs considerable cognitive burden. In practice, insufficient or coarse-grained inputs often result in degraded synthesis quality with weak stylistic expressiveness. To address this challenge, we propose AutoRich-TTS, a two-stage framework dedicated to automatically enriching simple user prompts into informative fine-grained style descriptions. Specifically, the first stage conducts context-driven implicit style cue extrapolation. The second stage generates enriched, fine-grained style descriptions based on the coarse input and the inferred contextual style cues, enhancing both the stylistic detail and expressive capacity of the prompts. Finally, an autoregressive speech language model (SLM) performs speech synthesis conditioned on both context and refined style prompts. Extensive subjective and objective evaluations verify that our approach effectively bridges the gap between vulgar user inputs and high-quality acoustic generation, achieving superior stylistic controllability while reducing user interaction costs.

Methodology

Overall Architecture of AutoRich-TTS

Automatic Prompt Enrichment

Our method enables the LLM to infer implicit context information and intent from the input text, dynamically adjusting the style of synthesized speech. Specifically, we use the LLM to generate Extrapolated Context surrounding sentences that encompass the topic and scenario of the Input Text. Then, a Coarse-grained Style Description serves as initial guidance, which combined with Extrapolated Context, helps generate Fine-grained Style Description with greater nuance and expressiveness. Finally, we pass the Extrapolated Context and the Input Text through the Context Compression module to obtain the Paragraph-level Semantic Prompt, which is combined with the Fine-grained Style Description to guide the TTS model to output expressive speech that meets user expectations.
