1. ABSTRACT
Conversational Speech Synthesis (CSS) aims to align synthesized speech with the emotional and stylistic context of user-agent interactions to achieve empathy. Current generative CSS models face interpretability limitations due to insufficient emotional perception and redundant discrete speech coding. To address the above issues, we present Chain-Talker, a three-stage framework mimicking human cognition: Emotion Understanding derives context-aware emotion descriptors from dialogue history; Semantic Understanding generates compact semantic codes via serialized prediction; and Empathetic Rendering synthesizes expressive speech by integrating both components. To support emotion modeling, we develop CSS-EmCap, an LLM-driven automated pipeline for generating precise conversational speech emotion captions. Experiments on three benchmark datasets demonstrate that Chain-Talker produces more expressive and empathetic speech than existing methods, with CSS-EmCap contributing to reliable emotion modeling. The code and demos are available at: https://github.com/Chain-Talker/Chain-Talker.

3.1 Chain-Talker Evaluation
We compare Chain-Talker with seven advanced CSS models using the NCSSD-EmCap dataset. These TTS models include:
1) Ground Truth: Real recorded speech.
2) CCATTS: A context-aware CSS model, which employs a GRU-based network to model the sentence-level dependency among the dialogue context (Guo et al., 2021).
3) M2-CTTS: It extracts context-dependent information at multiple granularities—both word-level and sentence-level—from the speech and text within the dialogue context (Xue et al., 2023).
4) ECSS: It considers different utterances in the dialogue context and modal information as individual graph nodes and uses a heterogeneous graph neural network to model the dialogue context (Liu et al., 2024a).
5) GPT-Talker: It takes the text and speech of the dialogue context as prompts and then utilizes a GPT-style autoregressive model to predict the discrete token sequence of the response speech, followed by synthesizing the final speech waveform using VITS (Liu et al., 2024b).
6) GPT-Talkerc: A variant of GPT-Talker adds empathetic captions during context modeling to help understand emotional changes in conversations.
7) Chain-Talkers: A variant of Chain-Talker replaces the empathetic captions with style factor labels.
8) Chain-Talkere: A variant of Chain-Talker replaces the empathetic captions with emotion labels.
9) Chain-Talker (Our Method): The approach we propose is a three-stage framework that mimics human cognition: Emotion Understanding derives context-aware emotion descriptors from dialogue history; Semantic Understanding generates compact semantic codes via serialized prediction; and Empathetic Rendering synthesizes expressive speech by integrating both components.

3.2 Ablation Experiment
To verify the effectiveness of our model components, we conduct ablation experiments by removing specific modules and training methods. These models include:
1) w/o context: A Chain-Talker variant without dialog history H, it predicts caption and renders speech directly from the given target utterance.
2) w/o captions: A CosyVoice variant, similar to GPT-Talker, where the input part includes various modal information from the dialogue history in addition to the target sentence, but without empathetic captions.
3) w/o Lcaption: Removing the loss function about emotion understanding.
4) w/o First-Stage: Training Chain-Talker directly on the NCSSD-EmCap dataset.

3.3 Verifying Emotion and Style Controllability
To assess the emotion and style control capability of Chain-Talker, we compare it against leading controllable TTS methods using both content and style prompts. These models include:
1) PromptTTS (Guo et al. 2023): This model uses prompts with style captions and content as input to synthesize corresponding speech, with the backbone model being Fast-Speech2.
2) Salle (Ji et al. 2024): This architecture treats style controllable TTS as a language model task, utilizing audio codec codes as an intermediate representation to replace the conventional mel-spectrogram.
3) Ground Truth: Real recorded speech.
